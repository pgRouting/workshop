1. Introducción¶

Resumen

pgRouting añade funcionalidad de ruteo a PostGIS. Este taller introductorio demostrará la funcionalidad de ruteo proporcionando ejemplos prácticos utilizando datos OpenStreetMap de la red de carreteras. Se tratarán temas que comienzan desde la preparación de los datos, la realización de consultas de enrutamiento, la escritura de funciones personalizadas “plpgsql” hasta la integración de pgRouting con otras herramientas FOSS4G.

La navegación por redes de carretera requiere algoritmos gráficos complejos. pgRouting es una biblioteca de código abierto extensible que proporciona una variedad de herramientas para algoritmos de gráficos, incluida la búsqueda de rutas más corta, como una extensión de PostgreSQL y PostGIS.

The workshop will focus on real road networks of the FOSS4G Firenze surrounding area. It will cover the following topics:

  • Instalación de pgRouting.

  • Creación de una topología de ruteo.

  • Importación de OpenStreetMap datos de la red de carreteras.

  • Uso de algoritmos pgRouting.

  • Escribir consultas avanzadas.

  • Escribir un procedimiento almacenado PostgreSQL personalizado en “plpgsql”.

Consulte contents para obtener todo el contenido de este taller.

Pre-requisitos

  • Nivel de taller: intermedio.

  • Conocimiento previo del asistente: SQL (PostgreSQL, PostGIS)

  • Equipments: This workshop uses OSGeoLive (15.0)
