1. Bilaga: Installation¶

För denna workshop behöver du:

  • Linux-operativsystem som Ubuntu

  • En redigerare som Gedit, Medit eller liknande

  • Geoserver för routing-applikationen

  • Internetanslutning

Installera pgRouting på en Windows-dator.

Följande referens är en snabb sammanfattning av hur du installerar det på din egen dator som kör Ubuntu 14.04 eller senare.

Ubuntu

pgRouting på Ubuntu kan installeras med hjälp av paket från ett PostgreSQL-arkiv:

Använda ett terminalfönster:

# Create /etc/apt/sources.list.d/pgdg.list. The distributions are called codename-pgdg.
sudo sh -c 'echo "deb https://apt.postgresql.org/pub/repos/apt/ $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'

# Import the repository key, update the package lists
sudo apt install wget ca-certificates
wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -
sudo apt update

# Install pgrouting based on your postgres Installation: for this example is 10
sudo apt install postgresql-10-pgrouting

Detta kommer också att installera alla nödvändiga paket som PostgreSQL och PostGIS om de inte är installerade ännu.

Att hålla sig uppdaterad om förändringar och förbättringar

sudo apt-get update & sudo apt-get upgrade

För att undvika fel med nekad behörighet för lokala användare kan du ställa in anslutningsmetod till trust i /etc/postgresql/<version>/main/pg_hba.conf och starta om PostgreSQL-servern med `sudo service postgresql restart.

Följ exemplet med PostgreSQL 10:

sudo nano /etc/postgresql/10/main/pg_hba.conf

local   all             postgres                                trust
local   all             all                                     trust
host    all             all             127.0.0.1/32            trust
host    all             all             ::1/128                 trust
